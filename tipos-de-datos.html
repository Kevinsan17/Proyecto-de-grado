Tipos de datos

En programación, un tipo de dato es la clasificación de un dato según sus características. Puede tratarse de una cadena, de un texto, de números, etc.

¡Para entender un poco mejor esto, puedes mirar el siguiente video!

Cuando construyes un programa, la materia prima que necesitas son los datos. Si pudieras ver muy de cerca la ejecución de dicho programa, podrías reconocer que, básicamente, lo que se hace es modificar, crear y leer datos.

Entre los tipos de datos y valores se encuentran:

  1. Numéricos (int): incluyen números positivos, negativos; cifras decimales, naturales, etc.
  2. Texto (string): letras, caracteres, símbolos que representan otros idiomas.
  3. Booleanos (boolean): que son fundamentales para establecer condiciones de verdadero y falso.
  4. Listas (array): para almacenar múltiples elementos de un mismo tipo.
